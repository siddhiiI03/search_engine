About This Project

Think Google, but just for your files.

This project is a web-based mini search engine that allows users to upload text files and search inside them using keywords.

Instead of manually opening multiple documents, users can simply type a word and quickly find the most relevant file.

The system is based on an inverted index for fast retrieval and a ranking mechanism to show the most useful results first.

Smart Search. Less Stress. More Productivity.
