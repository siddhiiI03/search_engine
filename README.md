# Findr - Web Based Mini Search Engine

Findr is a web-based document search engine developed as a minor project.
It allows users to upload text and PDF files, organize them into folders, and search content efficiently using inverted indexing and TF-IDF ranking.

## Features

* User Login / Signup
* Guest Mode Access
* TXT and PDF File Upload
* Folder Management
* Phrase Search
* Filename Search
* Stopword Removal
* Stemming
* TF-IDF Ranking
* PDF Export of Search Results

## Technologies Used

* Python
* Flask
* HTML
* CSS
* SQLite
* PyPDF2

## Search Pipeline

1. File Upload
2. Text Extraction
3. Preprocessing
4. Inverted Index Creation
5. TF-IDF Ranking
6. Ranked Search Results
